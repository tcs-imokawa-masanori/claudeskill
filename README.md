# Claude Skills

A curated list of top Claude Code skills for 2025.

## Official Anthropic Skills (Document Suite)

| Skill | Description |
|-------|-------------|
| **docx** | Create, edit, analyze Word docs with tracked changes, comments, formatting |
| **pdf** | Extract text, tables, metadata, merge & annotate PDFs |
| **pptx** | Read, generate, and adjust slides, layouts, templates |
| **xlsx** | Spreadsheet manipulation: formulas, charts, data transformations |

## Development & Testing Skills

| Skill | Description |
|-------|-------------|
| **Test-Driven Development** | Guides implementing features/bugfixes using TDD methodology |
| **Webapp Testing** | Tests local web apps using Playwright for UI verification and screenshots |
| **Subagent-Driven Development** | Dispatches independent subagents with code review checkpoints |

## Architecture & Workflow Skills

| Skill | Description |
|-------|-------------|
| **Software-Architecture** | Implements Clean Architecture, SOLID principles, and design best practices |
| **Superpowers** | Core skills library with 20+ battle-tested skills including TDD, debugging, and collaboration patterns |

## Integration & Utility Skills

| Skill | Description |
|-------|-------------|
| **MCP-Builder** | Guide for creating high-quality MCP servers to integrate external APIs |
| **Using-Git-Worktrees** | Creates isolated git worktrees with smart directory selection |
| **Skill Seekers** | Converts any documentation website into a Claude AI skill |

## Key Features of Skills

- **Portable**: Build once, use across Claude.ai, Claude Code, and API
- **Efficient**: Only loads what's needed (~100 tokens for metadata, <5k tokens when activated)
- **Model-invoked**: Claude autonomously decides when to use them based on your request

## Availability

Skills are available to **Pro, Max, Team, and Enterprise** users.
